# Business Name Generator

A simple JavaScript project that generates random business names by combining three different words: an adjective, a business category, and a suffix. The project demonstrates the use of conditional statements, variables, template literals, and the `Math.random()` function in JavaScript.

## 🚀 Features

* Generates unique business names randomly.
* Uses JavaScript's `Math.random()` function for randomness.
* Demonstrates the use of `if-else` conditions.
* Built without using arrays, making it a good beginner-friendly logic exercise.
* Produces names such as:

  * Crazy Engine Bros
  * Amazing Foods Limited
  * Fire Garments Hub

## 🛠️ Technologies Used

* JavaScript (ES6)

## 📋 How It Works

The program generates three random words:

### 1. Adjective

* Crazy
* Amazing
* Fire

### 2. Business Category

* Engine
* Foods
* Garments

### 3. Suffix

* Bros
* Limited
* Hub

A random number between 0 and 1 is generated using `Math.random()`. Based on the generated value, one word is selected from each category and combined to create a business name.
